# xping

## Install

### Linux / macOS
```bash
apt install git python3 python3-pip
git clone https://github.com/lemonaiddhater/xping
cd xping
pip3 install requests
python3 xping.py
```

### Termux
```bash
pkg update && pkg upgrade
pkg install git python
git clone https://github.com/lemonaiddhater/xping
cd ping
pip install requests
python xping.py
```

### iSH Shell
```bash
apk update
apk add python3 py3-pip git
git clone https://github.com/lemonaiddhater/xping
cd xping
pip3 install requests
python3 xping.py

```

### Windows
```powershell
winget install -e --id Python.Python.3; winget install -e --id Git.Git
python --version
git --version
git clone https://github.com/lemonaiddhater/xping
cd xping
python xping.py
```
